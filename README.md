## Installation & Running Locally
If you want to run the project locally:

```sh
# Clone the repository
git clone https://github.com/suryadeepkoduri/quote-machine.git
cd quote-machine

# Install dependencies
npm install

# Run the development server
npm run dev
```

## Contributing
